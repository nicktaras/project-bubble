# project-bubble


Project Bubble is an app for managing projects.

### Plan of Action

##### Create home page

Should contain:

    - Add project button
    - App title
    - Project tiles
    - Tiles should contain: image, title and status 
    
##### Create Project Page

Should contain:

    - Add tasks
    - Title
    - Three columns: not started, in progress and done
    - Progress / timeline bar
    - Tasks should be drag and dropable 
    
##### Add / Edit Task

Should contain inside a form:

    - Name
    - Developer
    - Time required / estimate 
    - Icons eg. bug, research etc. 
    - Status of task (default to not started)
    - Save / edit / delete / update button

##### Create Project Page

Should contain:

    - Edit
    - Add
    - Timeline / progress bar
    - Name of project
    - Project photo
    - Save / edit / delete
    
##### Create log in page

Should contain:

    - Sign in 
    - Sign up
    - Forgot password
